# Complaint Severity Prediction System

## Overview
This is a web-based application that analyzes user complaints and classifies them into severity levels (High, Medium, Low).

## Features
- User complaint input
- Severity prediction
- Simple UI
- Fast response

## Technologies Used
- HTML
- CSS
- JavaScript
- (Future) Machine Learning
- (Future) Google Gemini API

## Future Scope
- Integration of AI models (BERT + Random Forest)
- Improved prediction accuracy
- Real-time analytics dashboard
